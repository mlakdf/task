# Top 명령어는 시스템 상태를 파악 할수있는 명령어입니다.    (CPU,Memory,Process)

* 옵션 없이 사용하면 3초마다 화면을 갱신하며 정보를 보여줍니다

* top 명령어의 옵션
* -b(batch 모드) : 순간의 정보를 알려줌
* -n(반복횟수) : 기본 3초 주기를 -n 옵션으로 바꿀수 있음
